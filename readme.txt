Adds custom XML-RPC methods for using the wpmediamanager.com app.

== Description ==

This plugin just registers custom XML-RPC methods for using the [wpmediamanager.com] (http://wpmediamanager.com "wpmediamanager.com") app. At the moment it only includes a custom XML-RPC method to delete images.

== Installation ==

Simply drop the wp-media-manager-xmlrpc folder into your WordPress plugins directory and activate the plugin. That's it!
